

### Installation

    $ git clone https://github.com/jasonroelofs/slartibartfast.git
    $ git submodule init
    $ git submodule update
    $ source ./environment


    ## on OSX with homebrew
    # install FreeImage
    $ brew install FreeImage

    # install GLFW
    $ brew install GLFW

    # install   glew
    $ brew install  glew

    # compile and build
    $ rake build:all

    # run the game
    $ ./bin/slartibartfast
